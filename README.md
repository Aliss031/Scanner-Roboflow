# Roboflow Demo - Parcel Recipient Details

A web application that uses computer vision to detect parcel recipient details using Roboflow's inference engine.

## Features

- Real-time object detection using webcam
- FPS counter
- Responsive design

## Setup

1. Simply open `index.html` in a modern web browser
2. Allow camera access when prompted
3. The model will load and start detecting objects

## Technologies Used

- jQuery
- Lodash
- Async.js
- InferenceJS (Roboflow)

## Browser Compatibility

This application requires:
- A modern web browser with camera support
- JavaScript enabled
- Camera permissions
